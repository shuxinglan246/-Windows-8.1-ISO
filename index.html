      下载 Windows 8.1 光盘映像（ISO 文件）

      你转到此页面是因为你使用的操作系统不支持 Windows 8.1 媒体创建工具，我们希望确保你可以下载 Windows 8.1。 要使用媒体创建工具，请使用 Windows 7、Windows 8.1 或 Windows 10 设备访问 数星澜 Windows 8.1 页面。

      Windows8.1已停止支持，有关详细信息请参阅Windows 8.1 遵循固定生命周期策略

      开始之前

      • 确保你具有以下内容：
        • Internet 连接（可能产生 Internet 服务提供商费用）。
        • 在计算机、USB 或外部驱动器上有足够的数据存储可供下载。
        • 具有至少 6 GB 空间的空白 USB 或 DVD（和 DVD 刻录机）（如果你想创建介质）。我们建议使用空白 USB 或空白 DVD，因为其中如有任何内容都将被删除。
      • 阅读:https://www.microsoft.com/zh-cn/download/windows-usb-dvd-download-tool。
      • 阅读系统要求。
      • 如果你首次安装操作系统，将需要 Windows 产品密钥 (xxxxx-xxxxx-xxxxx-xxxxx-xxxxx)。有关产品密钥和何时需要它们的详细信息，请访问常见问题页面。
      • 有关企业版本，请访问批量许可服务中心。
      在你要安装 Windows 8.1 的 PC 上检查下列设置：
      • 64 位或 32 位处理器 (CPU)。你将创建适用于 CPU 的 64 位或 32 位版 Windows 8.1。若要在你的 PC 上检查此项，请转到“PC 设置”中的 PC 信息或“控制面板”中的系统，然后查找系统类型。
      • 安装 Windows 8.1 之前，请参阅系统需求。我们还建议你访问 PC 制造商的网站，了解有关更新的驱动程序和硬件兼容性的信息。
      • 安装 Windows 8.1 时，需要选择相同的语言。要了解你当前使用的语言，请转到 PC 设置中的时间和语言或控制面板中的区域。
      • 你还应该选择相同版本的 Windows。要检查你当前运行的版本，请转到 PC 设置中的 PC 信息或控制面板中的系统，并查找 Windows 版本。Windows 8.1 企业版不在此提供。请参阅批量许可服务中心了解详细信息。
